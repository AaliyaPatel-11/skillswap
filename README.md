# SkillSwap

SkillSwap is a peer-to-peer platform where users exchange skills instead of paying money. The platform connects individuals who want to teach a skill with others who want to learn it, creating a collaborative knowledge-sharing community.

## Overview

SkillSwap is built around the idea that knowledge should be accessible and collaborative. Instead of paying for courses or tutoring, users can trade the skills they already possess in exchange for learning something new.

Example:

* A user teaches **Graphic Design**
* In exchange for learning **Guitar**

The platform provides a structured environment where users can discover skills, request exchanges, communicate with matched users, and manage their learning profiles.

## Features

* User authentication with login and signup
* Landing page introducing the platform
* Dashboard displaying user activity and statistics
* Skill discovery page to browse available skills
* Skill swap request system
* Matches page listing connected users
* Chat interface for communication between matched users
* User profile management with editable skills
* Dark mode interface

## Getting Started

Clone the repository:

```bash
git clone https://github.com/AaliyaPatel-11/skillswap.git
```

Navigate into the project:

```bash
cd skillswap
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open in browser:

http://localhost:5173
